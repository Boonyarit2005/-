# 💧 Water Management System (WMS)
> ระบบบริหารจัดการน้ำประปาอัจฉริยะ เชื่อมโยงทุกภาคส่วนผ่านฐานข้อมูลเดียวเพื่อประสิทธิภาพสูงสุด

---

## 🧐 System Overview (ภาพรวมระบบ)
ระบบถูกออกแบบมาเพื่อรองรับผู้ใช้งาน 3 กลุ่มหลัก (Triple-User Architecture) โดยทำงานประสานกันผ่านศูนย์กลางข้อมูลเดียว:

* **👨‍💼 Admin (ผู้ดูแลระบบ):** ศูนย์กลางการจัดการบิลค่าน้ำ, บริหารข้อมูลพนักงาน/ลูกค้า และมอบหมายงานซ่อม (Maintenance Dispatch).
* **👮 Officer (เจ้าหน้าที่):** ระบบติดตามคุณภาพน้ำ (ความขุ่น/คลอรีน) แบบ Real-time, รับงานซ่อมตามโซนพื้นที่ และระบบแผนที่นำทาง.
* **👤 Customer (ลูกค้า):** ตรวจสอบบิลค่าน้ำ, ชำระเงินผ่าน QR Code และแจ้งซ่อมด้วยระบบปักหมุดพิกัด (GPS Pinning).

## ✨ Brand Color Palette (ธีมสีของระบบ)
เราเลือกใช้สีที่สื่อถึงความสะอาด ความน่าเชื่อถือ และความปลอดภัย:

| สี | ความหมาย | การใช้งานในระบบ |
| :--- | :--- | :--- |
| **Sky Blue (ฟ้า)** | น้ำและการบริการ | Header, ปุ่มยืนยันหลัก, Brand Identity |
| **Success Green (เขียว)** | ถูกต้อง / ปลอดภัย | ปุ่มออกบิล, รับงานซ่อม, สถานะคุณภาพน้ำดี |
| **Alert Orange/Red (ส้ม/แดง)** | ตรวจสอบ / เร่งด่วน | แจ้งเตือนยอดค้างชำระ, ค่าความขุ่นน้ำสูง, แจ้งซ่อม |
| **Clean Gray (เทาอ่อน)** | สบายตา / โปร่งใส | พื้นหลัง Dashboard, ตารางข้อมูล |

## 🛠 Key Features (จุดเด่นของระบบ)

* **Real-time Dashboard:** สรุปปริมาณการใช้น้ำและคุณภาพน้ำ (Turbidity/Chlorine) ทันทีในหน้าแรก
* **End-to-End Billing:** ระบบออกบิลออนไลน์ที่รองรับการแนบสลิปเพื่อยืนยันการชำระเงินอัตโนมัติ
* **Smart Maintenance:** ระบบแจ้งซ่อมที่ระบุพิกัดชัดเจน ช่วยให้เจ้าหน้าที่เข้าถึงหน้างานได้รวดเร็ว
* **Data Consistency:** ทุกกลุ่มผู้ใช้มองเห็นข้อมูลชุดเดียวกัน ลดความซ้ำซ้อนของข้อมูล
